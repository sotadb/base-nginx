Welcome to NGiNX, SotADB edition

If you're reading this, NGiNX is setup and running. It's configured to serve static content, unencrypted to port 8080. This is intended to be used behind Kubernetes' ingress server, which should also provide the SSL endpoint. All security setting should be provided there.

This particular configuration was created to serve as a foundational container for www.sotadb.info. It has been made public, so feel free to use it. If you have any observations about how it can be improved (particularly how it's security can be improved) please post an issue to our github issues page.
