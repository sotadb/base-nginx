Welcome to NGiNX, SotADB edition

If you're reading this, NGiNX is setup and running. It's configured for SSL, and pulled an SSL cert from Let's Encrypt. "Export" Cypher Suites have been disabled, Elliptic Curve Diffie-Hellman Key Exchange has been enabled, and a safe, strong, 2048-bit prime Diffie-Hellman group was generated. This can be tested with Qualsys' SSL Server Test.

With the exception of some management pages, CSP is enabled, and set to "self". This instructs web-browser to deny loading inline scripts or resources from external sites, and possibly will need overriden.

This particular configuration was created to serve as a secure foundational container for www.sotadb.info. It has been made public, so feel free to use it. If you have any observations about how it can be improved (particularly how it's security can be improved) please post an issue to our github issues page.
